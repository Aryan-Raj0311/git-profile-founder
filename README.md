# git-profile-founder 🚀
A responsive GitHub Profile Finder that displays user information and creates QR codes for easy profile sharing. 

A simple web application that allows users to search for GitHub profiles, view public account information, and generate a QR code for quick profile access.

## Features

- 🔍 Search GitHub users by username
- 👤 Display profile avatar and username
- 📊 View public repositories, followers, and following count
- ✅ Username validation
- 📱 Generate a QR code linked to the user's GitHub profile
- ⚠️ Handle invalid usernames and API errors gracefully

## Tech Stack

- HTML5
- Tailwind CSS
- JavaScript (ES6)
- GitHub REST API
- QRCode.js

## How It Works

1. Enter a GitHub username.
2. The application fetches user data from the GitHub API.
3. Profile information is displayed dynamically.
4. A QR code is generated for the user's GitHub profile URL.
5. Scan the QR code to open the profile directly.

## Installation

```bash
git clone <repository-url>
cd github-profile-finder
```

Open `index.html` in your browser.
